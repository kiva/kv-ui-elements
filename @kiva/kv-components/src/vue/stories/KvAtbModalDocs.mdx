# KvAtbModal

A contextual modal component that provides feedback and engagement opportunities after a user adds a loan to their basket, displaying milestone progress, achievement notifications, and personalized messaging.

## Component Overview

The Add-to-Basket (ATB) Modal enhances the lending experience by providing immediate visual feedback when users add loans to their basket. It dynamically adapts its content based on the user's lending history, milestone progress, and proximity to achievements, encouraging continued engagement.

## Variations

## Usage Information

The KvAtbModal provides contextual feedback after basket additions, encouraging users to continue their lending journey through gamification and achievement tracking.

### When to Use

- After a user successfully adds a loan to their basket
- To provide immediate visual confirmation of the basket action
- To display and encourage milestone/achievement progress
- To highlight when users are close to unlocking achievements
- To celebrate goal completion moments

### When Not to Use

- For general modal dialogs unrelated to basket actions
- For error messages or critical alerts
- When the basket action failed and needs error handling
- For checkout or payment flow modals

## Behavior

### Modal Positioning

The modal positions itself relative to the header basket element, appearing below and aligned with the basket icon. It uses CSS custom properties (`--modal-right`, `--modal-top`) to dynamically adjust its position based on scroll and resize events.

<Story of={KvAtbModalStories.Default} />

### Content Adaptation

The modal intelligently adapts its content based on the user's context:

1. **Guest/First-time user**: Shows welcome badge and personalized message
2. **Returning user**: Displays milestone progress and achievement notifications
3. **Close to achievement**: Highlights proximity with "one away" messaging
4. **Goal tracking**: Shows annual goal progress or completion celebration

<Story of={KvAtbModalStories.MilestoneAchievement} />

## Anatomy

The modal consists of the following structural elements:

- **Modal Container** (`KvCartModal`): Outer wrapper providing positioning and background
- **Borrower Photo**: Visual representation of the funded loan
- **Content Section**: Contains the pill message and relevant badges
- **Achievement Pill** (`KvCartPill`): Displays milestone info, borrower name, and custom messaging
- **Icon Slot**: Shows contextual icons (first loan badge, achievement indicator)
- **Action Buttons**: View basket and support another loan options (via `KvCartModal`)

---

## Specs

### Positioning

The modal is positioned using CSS custom properties that are recalculated on scroll and resize:

- `--modal-right`: Distance from the right edge of the viewport
- `--modal-top`: Distance from the top of the viewport (below header)

### Responsive Behavior

- On desktop (`md` and above): Modal anchors to the header basket element
- Position updates are throttled (50ms) for performance during scroll/resize

## Accessibility

- **Focus management**: Modal traps focus appropriately when visible (handled by `KvCartModal`)
- **Keyboard navigation**: Supports Escape key to close the modal
- **Screen reader announcements**: Content changes are announced appropriately
- **Color contrast**: All text and interactive elements meet WCAG contrast requirements
- **Touch targets**: Buttons meet minimum 44x44px touch target requirements

## Code Examples

### Basic Usage

```vue
<template>
  <header>
    <div data-testid="header-basket">
      <KvAtbModal
        :modal-visible="showModal"
        :added-loan="addedLoan"
        :user-data="userData"
        photo-path="https://www.kiva.org/img/"
        :show-modal-content="true"
        @reset-modal="handleResetModal"
        @close-redirect="handleCloseRedirect"
      />
    </div>
  </header>
</template>

<script setup>
import { ref } from 'vue';
import { KvAtbModal } from '@kiva/kv-components';

const showModal = ref(true);
const addedLoan = {
  id: 12345,
  name: 'Maria',
  gender: 'female',
  borrowerCount: 1,
  themes: [],
  basketSize: 1,
};
const userData = {
  my: {
    loans: { totalCount: 5 },
  },
};

const handleResetModal = () => {
  showModal.value = false;
};

const handleCloseRedirect = ({ type, path }) => {
  showModal.value = false;
  // Navigate to path based on type
};
</script>
```

### One Loan Away Achievement

```vue
<template>

};
</script>
```
